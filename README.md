# TP Liste des tâches

Gestionnaire de tâches web simple en Vue.js

## Thème choisi : Gestionnaire de tâches façon Trello
J'ai choisi ce thème plutôt que le gestionnaire de quêtes RPG car je voulais faire un projet 
professionnel, épuré et neutre, plus facilement utilisable par un plus grand nombre
de personnes.
Mais le thème RPG reste tout de même très intéressant. Donc, au final je dirais que j'ai choisi ce 
thème de peu.

## Fonctionnalités
- Gestion des tâches : ajouter, modifier et supprimer
- Stockage : tâches sauvegardées en local
- Tableau de bord : Visualisation orgnaisée des tâches par statut (Toutes, à faire, en cours, terminée)

- Recherche dynamique : filtrage des tâches par titre ou description
- Filtrage par priorité
- Thèmes : mode clair/sombre
- Statistiques : affichage du nombre de tâches par statut

## Installation et lancement

1. Cloner le projet
git clone https://github.com/Mahereiti/TP_gestionnaire_taches

2. Installer les dépendances
npm install

3. Lancer le serveur
npm run dev
